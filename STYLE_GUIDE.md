# 📝 章节写作风格指南

## 风格要求

所有生成的章节必须严格按照 `example.md` 中的风格来写。

## 核心风格特点

### 1. 标题格式

**正确格式：**
```
第一章：抢戏的配角们——诸侯不"侯"，天子在打酱油
```

**错误格式：**
```
# 第一章：抢戏的配角们——诸侯不"侯"，天子在打酱油
```

- ❌ 不要使用Markdown标题符号（#、##等）
- ✅ 直接写标题，保持简洁

### 2. 开篇风格

**必须使用：**
```
各位看官，且泡上一壶好茶，咱们的穿越之旅这就启程。
```

**特点：**
- 使用"各位看官"称呼读者
- 用"且泡上一壶好茶"这样的开场
- 营造对话感，像在跟朋友聊天
- 使用"您"来称呼读者

### 3. 语言风格

#### 现代比喻

大量使用现代商业、职场比喻：
- "董事长"、"分公司总经理"、"CEO"
- "KPI"、"公司"、"总部"、"上市"
- "内卷"、"霸凌"、"烂活"

#### 网络用语

使用网络流行语和现代语言：
- "作死"、"内卷"、"霸凌"
- "烂活"、"废柴"、"戏精"
- "嗑瓜子"、"撸袖子"

#### 括号注释

使用括号来解释和补充：
- "这叫'进贡'"
- "史称'诸侯'"
- "（如晋文侯、郑武公）"

### 4. 结构要求

#### 分节格式

使用"一、二、三"格式：
```
一、 一场狼狈的搬家，拉开了大戏的帷幕

内容...

二、 第一个掀桌子的"刺头"：郑庄公

内容...

三、 "霸主体制"下的花式内卷

内容...
```

#### 结语格式

必须有"本章结语："部分：
```
本章结语：
所以你看，春秋这出大戏，主角从来不是台上那位唯唯诺诺的"董事长"...

好了，茶凉了，咱们第一章就先讲到这儿。下一章，咱们将进入一个更刺激的时代...
```

### 5. 图片格式

**必须使用markdownURL格式：**


**特点：**
- 单独一行
- 使用Markdown格式 `![描述](路径)`
- 图片插入在段落之间，与内容相关

**图片插入位置：**
- 开篇后插入一张主题图片
- 每个分节可以插入相关图片
- 结语前可以插入总结性图片
- 每章3-6张图片

### 6. 内容要求

#### 字数
- 控制在4000-6000字

#### 内容结构
1. **开篇引子**：用核心梗的比喻引入，风趣幽默
2. **分节内容**：使用"一、二、三"格式，每个分节讲述一个主题
3. **历史事件**：按时间顺序介绍，用现代语言描述
4. **风云人物**：介绍关键人物，用现代比喻
5. **历史趣闻**：穿插有趣的小故事
6. **本章结语**：总结本章，引出下一章

#### 语言特点
- 风趣幽默，口语化
- 使用现代比喻和网络用语
- 保持历史事实准确
- 营造对话感

- [ ] 有"本章结语："部分

- [ ] 大量使用现代比喻和网络用语
- [ ] 使用"您"称呼读者
