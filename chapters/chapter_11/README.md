第十一章：“内斗”比“外敌”更可怕——司马家的“兄弟阋墙”

![司马炎登基](https://img2.baidu.com/it/u=2685825005,1107231483&fm=253&fmt=auto&app=120&f=JPEG?w=1257&h=800)

各位看官，且泡上一壶上好的龙井，咱们今天要聊的这段历史，堪称是中国古代“家族企业”管理失败的经典案例。话说上一回，曹魏集团的“职业经理人”司马家族，经过三代人（司马懿、司马师、司马昭）的苦心经营，终于把董事长曹家的股份全给“收购”了。到了公元265年，司马昭的儿子司马炎，觉得时机成熟，正式让曹魏末代皇帝曹奂“禅让”，自己登基称帝，改国号为“晋”，史称西晋。这感觉，就像一家公司，三代CEO（首席执行官）卧薪尝胆，终于把创始人家族彻底踢出局，自己当上了董事长兼大股东，完成了“借壳上市”。司马炎，就是这西晋集团的第一任、也是唯一一任真正说了算的董事长。

一、 上市即巅峰：司马炎的“天下一统”KPI

司马炎接手这家新公司时，家底相当厚实。爷爷司马懿熬死了诸葛亮，耗死了曹魏的忠臣；伯伯司马师和爸爸司马昭，更是把曹魏内部清理得干干净净，还顺手灭了意图独立的蜀汉分公司（263年）。所以，司马炎上任后的首要KPI（关键绩效指标），就剩下兼并最后一个竞争对手——东吴集团了。

这位晋武帝司马炎，初期表现堪称“别人家的老板”。他吸取了曹魏皇室孤立无援的教训，一口气封了二十七个同姓王爷（基本都是他的叔伯、兄弟、子侄），让他们掌握实权，分镇各地。这思路很“朴素”：曹家就是因为没自己人帮忙，才被我们司马家“和平演变”了。所以，咱们得把自家兄弟子侄都安排到重要岗位上，这叫“肥水不流外人田”，家族企业就得用自家人，靠谱！

![三国归晋图](https://img0.baidu.com/it/u=1670426872,2219462922&fm=253&fmt=auto&app=120&f=JPEG?w=1141&h=800)

为了完成统一大业，司马炎也是励精图治。他发展生产，出现了“太康之治”的繁荣景象（太康是他的年号）。人口增加了，国库也充实了。然后，就是等待时机。东吴那边的末代老板孙皓，是个著名的“暴君+昏君”结合体，搞得公司上下怨声载道，人才流失严重。司马炎一看，这简直是天赐良机。

公元279年，司马炎一声令下，六路大军水陆并进，大举伐吴。东吴那边已经是“摆烂”状态，防线一触即溃。第二年（280年），晋军攻入建业（今南京），孙皓投降。至此，自东汉末年黄巾之乱开始，将近百年的分裂割据局面，终于画上了句号。司马炎完成了统一，西晋集团成功“垄断”了整个中国市场，股价（国势）达到了巅峰。

如果故事在这里结束，那司马炎绝对是千古一帝的有力竞争者。但可惜，历史没有如果。统一之后，司马炎和他庞大的司马家“董事会”，就开始了一系列令人瞠目结舌的“骚操作”。

二、 埋雷高手：司马炎的“神操作”与“傻白甜”太子

公司上市，成为行业垄断巨头后，司马炎董事长的心态就有点飘了。他开始追求“生活品质”。史书记载他后宫妃嫔超过万人，自己都挑花眼了，于是发明了“羊车望幸”的著名桥段——坐着羊拉的小车在后宫转悠，羊停在哪个妃子门口，就临幸哪位。您瞧瞧，这哪是皇帝，这分明是“随机抽奖式临幸”的开创者。上行下效，整个统治集团迅速腐化，斗富成风。大臣石崇和王恺斗富的故事，什么用蜡烛当柴火烧，用锦缎做五十里步障，拿胡椒涂墙，都是这个时期的“炫富名场面”。公司高层如此，中下层管理干部（官僚）的风气也就可想而知了。

但比起生活腐化，司马炎在“继承人问题”上埋的雷，才是真正致命的。

他的太子叫司马衷，就是历史上著名的晋惠帝。这位爷的智商，在史书里留下了两个千古笑谈。一是“何不食肉糜”？天下荒乱，百姓饿死，他惊奇地问：“没有粮食，为什么不吃肉粥呢？”二是“青蛙为谁叫”？他在华林园听到蛤蟆叫，问左右：“这叫声，是为公事，还是为私事？”左右憋着笑回答：“在公家地里的为公，在私人地里的为私。”

![晋惠帝司马衷](https://img2.baidu.com/it/u=1092071677,202425432&fm=253&fmt=auto&app=138&f=JPEG?w=608&h=681)

就这么一位“傻白甜”太子，司马炎难道不知道吗？他知道，但他有他的“苦衷”和“侥幸心理”。司马衷是嫡长子，母亲是皇后杨艳，背后是势力庞大的弘农杨氏。而且，司马衷有个聪明儿子叫司马遹（yù），小时候就被司马炎夸赞像自己的爷爷司马懿。司马炎想的是：儿子虽然笨点，但孙子聪明啊！我先把位子传给儿子，等孙子长大了，不就能接个好班了吗？这叫“隔代指定接班人”。

为了让傻儿子坐稳江山，司马炎又做了两件“配套”的蠢事。第一，他给太子娶了个极其强悍的媳妇——贾南风。贾南风是开国功臣贾充的女儿，长得又黑又矮又丑，但性格嫉妒、狡诈、凶悍，权力欲极强。司马炎本来不同意，但架不住皇后和权臣的劝说。这就等于给一个智商不高的董事长，配了一个野心勃勃、控制欲超强的“董事长夫人兼实际CEO”。

第二，为了制衡可能出现的权臣（包括儿媳妇），司马炎更加倚重自家那些手握重兵的王爷兄弟们。他以为，血缘是最可靠的纽带。这些王爷有兵有权有地盘，一旦中央有难，他们就是“勤王”的保障。但他完全没想过，如果这些王爷自己就是“难”的来源呢？

公元290年，司马炎在埋下无数隐患后撒手人寰。傻儿子司马衷即位，精明强悍的丑媳妇贾南风成为皇后。西晋这家刚刚完成垄断的“家族企业”，正式交到了一个“低能董事长”和一个“悍妇CEO”手里，而周围，是一群虎视眈眈、手握实权的“大股东”（王爷们）。火药桶的引线，已经点燃。

三、 导火索：“悍妇CEO”贾南风的权力游戏

司马衷上台后，朝政大权很快落到了皇后贾南风手里。这位贾皇后，可不像她的傻丈夫。她有着敏锐的政治嗅觉和狠辣的手段。但她面临一个障碍：当时的辅政大臣是太傅杨骏（杨太后的父亲，也是外戚）。杨骏想独揽大权，把贾南风晾在一边。

贾南风哪能受这气？她立刻想到了司马炎留下的“遗产”——那些兵强马壮的王爷们。于是，她秘密联络了镇守荆州的楚王司马玮（司马炎的儿子，司马衷的弟弟），让他带兵进京“清君侧”。公元291年，司马玮带兵进洛阳，杀了杨骏及其党羽，废了杨太后。这是“八王之乱”的第一枪。

![贾南风弄权](https://img2.baidu.com/it/u=2543073960,73353966&fm=253&fmt=auto&app=120&f=JPEG?w=1280&h=800)

但贾南风很快发现，请神容易送神难。杀了杨骏后，朝政又被汝南王司马亮（司马懿的儿子，司马衷的叔祖）和元老卫瓘把持。司马亮辈分高，声望大，贾南风还是觉得碍手碍脚。于是，她再次施展“借刀杀人”之计。她让傻皇帝司马衷写密诏给楚王司马玮，说司马亮和卫瓘要谋反，令其诛杀二人。司马玮是个愣头青，真就带兵把德高望重的司马亮和卫瓘给杀了。

事情办完后，贾南风立刻翻脸不认账，以“擅杀大臣”的罪名，处死了楚王司马玮。一招“连环计”，贾南风一举除掉了杨骏、司马亮、卫瓘、司马玮四股势力，自己独揽大权。这段时间（291-299年），贾南风任用亲信，虽然她本人荒淫残暴，但朝政大体还算平稳，甚至史书称“朝野安静”。她似乎证明了，哪怕董事长是傻子，只要CEO足够强悍，公司也能勉强运转。

然而，贾南风心里始终有个疙瘩，那就是太子司马遹。司马遹不是她亲生的（是谢才人所生），而且聪明英武，很得人心。贾南风自己没儿子，她害怕将来司马遹即位，会清算自己。于是，恶向胆边生。公元299年，贾南风设计诬陷太子司马遹谋反，将他废为庶人，后又派人将其杀害。

这下，可捅了马蜂窝了。太子是国本，你贾南风再厉害，杀太子就是动摇国本，给了所有野心家最完美的借口。一直对贾南风专权不满的赵王司马伦（司马懿的儿子，司马衷的叔祖），和他的谋士孙秀，抓住了这个机会。

四、 全面开撕：“八王”轮番登场的大乱斗

赵王司马伦早就有野心，但一直畏惧贾南风。现在太子被杀，人心沸腾，他觉得时机到了。公元300年，司马伦率兵入宫，以“为太子报仇”的名义，矫诏逮捕并毒死了贾南风，顺便把她的一干党羽全部清洗。

但司马伦的野心不止于此。第二年（301年）正月，他干脆逼傻皇帝司马衷“禅让”，自己登基做了皇帝。这下，可把其他王爷给惹毛了。大家原本以为你司马伦是“勤王功臣”，没想到你是个“篡位贼子”！你能当皇帝，我为什么不能？

于是，大乱斗正式升级。镇守许昌的齐王司马冏（司马攸的儿子，司马炎的侄子）、镇守邺城的成都王司马颖（司马炎的儿子，司马衷的弟弟）、镇守关中的河间王司马颙（司马懿弟弟的孙子，司马衷的族叔）三方联合起兵，讨伐司马伦。洛阳的禁军将领也倒戈响应。司马伦众叛亲离，很快兵败被杀，傻皇帝司马衷又被迎回来复位。齐王司马冏因功劳最大，入朝辅政。

__八王之乱形势图__

但权力就像毒药，尝过滋味的人都不想放手。司马冏辅政后，也开始骄奢专权，引起了其他王爷的不满。302年，河间王司马颙联合在洛阳的长沙王司马乂（司马炎的儿子，司马衷的弟弟），里应外合讨伐司马冏。司马乂在洛阳城内动手，杀了司马冏，自己掌控了朝政。

司马颙本来想利用司马乂，没想到司马乂自己掌了权，于是又联合成都王司马颖，发兵攻打洛阳。司马乂虽然兵力处于劣势，但很能打，在洛阳坚守了几个月。关键时刻，在朝中的东海王司马越（司马懿弟弟的孙子，司马衷的族叔）害怕城破遭殃，发动政变，逮捕了司马乂，并将其交给司马颙的部将张方。张方用火将司马乂活活烤死。

至此，成都王司马颖成了最大赢家，他被封为皇太弟（皇帝的接班人），遥控朝政。但东海王司马越又不服了。304年，司马越带着傻皇帝司马衷，发兵讨伐在邺城的司马颖，结果在荡阴（今河南汤阴）大败。混战中，司马衷脸上中箭，身陷敌营，成了“肉票皇帝”。司马越逃回自己的封国。

螳螂捕蝉，黄雀在后。一直伺机而动的河间王司马颙，趁机派部将张方占领洛阳。并州刺史司马腾（司马越的弟弟）和幽州刺史王浚，联合北方的鲜卑、乌桓骑兵，南下攻打司马颖。司马颖战败，带着皇帝司马衷逃往洛阳，落入张方手中。张方又挟持皇帝和司马颖西迁长安，长安是司马颙的地盘。

到了长安，司马颙废了司马颖的皇太弟之位，自己独揽大权。但他也控制不住局面。305年，东海王司马越再次起兵，联合各方势力西攻关中，讨伐司马颙。经过一年混战，司马越获胜。306年，司马颙、司马颖相继被杀。傻皇帝司马衷被司马越迎回洛阳。同年，司马衷神秘死亡（普遍认为是司马越毒杀），司马越立司马衷的弟弟司马炽为帝，是为晋怀帝。司马越独掌大权。

从291年贾南风引发楚王司马玮杀杨骏，到306年司马越最终胜出，这场持续了十六年的“八王之乱”才勉强落下帷幕。参与核心混战的王爷主要有八个：汝南王司马亮、楚王司马玮、赵王司马伦、齐王司马冏、长沙王司马乂、成都王司马颖、河间王司马颙、东海王司马越。（“八王”具体名单有不同说法，但核心是这几位）。他们像走马灯一样轮流登场，互相攻杀，把中原大地变成了修罗场。

五、 自毁长城：内战如何掏空帝国根基

这场司马家内部的“超级内卷”，其破坏力是毁灭性的，远超过任何外部敌人。

首先，军队打没了。王爷们为了取胜，不仅动用自己封国的军队，还大量引进外援。比如成都王司马颖引匈奴刘渊为外援，东瀛公司马腾和幽州的王浚引鲜卑、乌桓骑兵参战。这些胡族军队在中原烧杀抢掠，形同匪寇。而中央的精锐禁军和各地驻军，在无休止的内战中消耗殆尽。帝国的国防力量被自己人亲手瓦解。

其次，经济打崩了。战火主要集中在中原核心区（河南、河北、关中），这些原本最富庶的地方，城市被反复洗劫，农田荒芜，百姓要么死于战乱，要么流离失所。《晋书》记载：“百姓流亡，中原萧条，千里无烟。” 国家的财政税收体系完全崩溃。

![永嘉之乱](https://img0.baidu.com/it/u=2891748084,736446524&fm=253&fmt=auto&app=138&f=JPEG?w=560&h=500)

再者，人才和人心打散了。许多有识之士看到国家无望，纷纷南渡避祸，为后来的东晋留下了人才种子。而留在北方的士族和百姓，对司马氏政权彻底失望。当朝廷无法保护你，反而成为灾难的来源时，它的合法性就荡然无存了。

最后，也是最重要的，他们把“狼”引进了家门。为了在内斗中取胜，王爷们竞相拉拢边境的胡族势力，给予他们官职、武装他们。匈奴的刘渊、羯族的石勒，这些日后的“乱华”主角，最初都是在“八王之乱”的战场上为司马家的王爷们打工，学会了中原的军事技术和政治权谋，看清了西晋外强中干的本质。

六、 最后的崩塌：“五胡”入场与“永嘉之耻”

当司马越在307年终于“熬死”所有主要竞争对手，独揽朝纲时，他面对的已经是一个千疮百孔、奄奄一息的烂摊子。中央权威扫地，地方割据四起，而北方边疆的胡族势力已经羽翼丰满。

早在304年，“八王之乱”正酣时，匈奴贵族刘渊就在并州左国城（今山西离石）起兵，自称汉王，打出了复兴“汉朝”的旗号（他自称是汉朝公主的后裔，姓刘）。这不是偶然，他看到司马家骨肉相残，中原空虚，知道机会来了。

羯族人石勒，原本是被人贩卖的奴隶，后来加入农民起义军，再后来投靠了司马颖的部将，在战乱中逐渐拉起一支凶悍的军队。他就像一头在血腥战场上成长起来的猛兽，战斗力极强，而且对晋朝贵族充满仇恨。

![刘渊建汉](https://img2.baidu.com/it/u=3416573422,2261363467&fm=253&fmt=auto&app=138&f=JPEG?w=800&h=1088)

311年，司马越在忧惧中死于项城（今河南沈丘）。他带领的朝廷最后一点精锐军队和大量随军朝臣，在护送其灵柩回东海封国的途中，被石勒的骑兵在苦县宁平城（今河南郸城）追上并包围。一场血腥屠杀后，西晋最后的中央军队和包括王公大臣在内的十多万随行人员全军覆没，史称“宁平城之难”。晋朝的核心统治集团被一锅端。

同年，刘渊的儿子刘聪（此时国号已改为“赵”，史称前赵）派大将刘曜、石勒等攻破西晋都城洛阳。晋怀帝司马炽被俘，皇宫被焚，官员百姓被杀三万余人，珍宝文物被洗劫一空。这就是历史上著名的“永嘉之乱”（永嘉是晋怀帝的年号）。怀帝被掳到平阳（今山西临汾），受尽侮辱，最终被杀。

洛阳陷落后，晋朝大臣们在长安拥立晋怀帝的侄子司马邺为帝，是为晋愍帝。但长安也是孤城一座，饥荒严重，“人相食”。316年，刘曜再次率军围攻长安，晋愍帝在粮尽援绝的情况下，肉袒出降，西晋正式灭亡。愍帝同样被掳到平阳，受尽屈辱后被杀。

从265年建国，到316年灭亡，西晋国祚仅51年。如果从280年统一全国算起，其统一时间只有短短36年。真正是“其兴也勃焉，其亡也忽焉”。而随着西晋中央政权的崩溃，北方陷入了长达一百多年的“五胡十六国”大混战时期（匈奴、羯、鲜卑、氐、羌等族先后建立政权），社会遭到空前破坏，这是中国历史上的一段黑暗时期。

本章结语：
回看西晋这段历史，简直是一部标准的“作死”教科书。董事长司马炎，好不容易兼并所有对手，完成行业垄断上市，却因为错误的“家族信托”方案（大封同姓王）和选了一个“败家”的继承人，给公司埋下了超级火药桶。接着，悍妇CEO贾南风为了个人权欲，点燃引信。然后，司马家那群手握重兵的“大股东”们，为了争夺公司控制权，开始了长达十六年的血腥“内部收购战”。他们把公司核心资产（军队、财力、人才、民心）全部打光，把公司总部（中原）砸得稀巴烂。最后，当他们自己都精疲力尽、奄奄一息时，早就被他们请来当“临时工”甚至“打手”的“野蛮人”（胡族势力），轻而易举地闯进来，宣布公司破产清算，瓜分了一切。

它用血淋淋的事实告诉我们一个道理：对于一个组织、一个国家而言，很多时候，最可怕的敌人不是外部的挑战者，而是内部无休止的倾轧与内耗。堡垒最容易从内部攻破。司马家拥有统一后的巨大优势，却因为无法解决权力分配和继承的核心矛盾，上演了一出“亲者痛、仇者快”的悲剧，不仅让自己家族遭受灭顶之灾，更将整个北中国拖入了深渊。所谓“八王之乱”，乱的何止是八个王，乱的是天下，苦的是苍生。这段历史留给后人的，除了唏嘘，更应是深深的警醒。
