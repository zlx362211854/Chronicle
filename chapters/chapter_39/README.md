第三十九章："军阀混战"的"时代"——民国乱世的"天下还是乱的"

![袁世凯就任大总统](https://img1.baidu.com/it/u=3746900055,1693816933&fm=253&fmt=auto&app=120&f=JPEG?w=805&h=500)

各位看官，且泡上一壶浓茶，咱们上一章还沉浸在辛亥革命的狂欢中，觉得中国总算要迎来新纪元了。但这一章您可得坐稳了，因为咱们要进入的，是一个比春秋战国还要混乱的"超级内卷时代"——民国北洋时期。这哪是什么"民国"，分明是一家刚刚完成IPO（上市）就立刻陷入管理层恶斗、股东内讧、分公司纷纷独立的"崩溃式集团重组现场"。董事长换了，但天下，嘿，还是乱的。

一、 从"临时CEO"到"终身皇帝"：袁世凯的83天皇帝梦

话说1912年，大清这家"百年老字号"终于破产清算，新公司"中华民国有限公司"挂牌成立。按照投资协议（也就是《清帝退位优待条件》），原大清集团的总经理袁世凯，凭借手握北洋新军这支"核心业务团队"，成功转型为新公司的"临时大总统"——您可以理解为临时CEO。

这位袁CEO一开始还挺像那么回事。他穿着西装，喊着"共和"，跟各路股东（革命党、立宪派、外国资本）都保持良好关系。但很快，问题来了：这位CEO骨子里还是个封建大家长啊！他觉得公司这么搞"民主董事会制度"太麻烦了，今天这个股东要发言权，明天那个董事要查账，哪有当年在大清集团说一不二来得爽？

于是，他开始了一系列"骚操作"。

第一步，打压"董事会"（国会）。1913年，他派人暗杀了主张责任内阁制的国民党领袖宋教仁（这就好比现代公司里CEO派人干掉了主张分权的董事会主席），接着武力镇压了国民党发起的"二次革命"，把反对派股东们要么赶走，要么打趴。

第二步，修改"公司章程"（《中华民国约法》）。1914年，他炮制了一部《中华民国约法》，把权力全部集中到"大总统"手里，任期十年，还能连任，甚至可以指定继承人。好家伙，这哪是总统，这分明是"终身董事长兼CEO"啊！他还解散了国会，把议员们像赶苍蝇一样赶回老家。__袁世凯祭天__

但袁CEO的野心还不止于此。他觉得，"董事长"哪有"皇帝"来得名正言顺、威风八面？他身边还聚集了一帮"劝进团"，整天给他灌迷魂汤，说什么"天命所归"、"民众拥戴"，甚至还搞出了"乞丐请愿团"、"妓女请愿团"这种奇葩组织，上演了一出出"全民跪求袁老板登基"的荒诞剧。

最绝的是，他那个在美国留过学、本该最懂共和的儿子袁克定，为了当太子，居然伪造了一份《顺天时报》（当时的主流报纸），天天只登"日本支持帝制"、"万民盼君"的假新闻给他老爹看。这就好比现代儿子为了骗老爸的钱，天天PS一些"全球股市暴涨，就等您老出山"的假新闻。

于是，在1915年12月12日，袁世凯正式宣布接受"国民推戴"，改国号为"中华帝国"，年号"洪宪"，准备在1916年元旦登基。

结果呢？这场皇帝梦，只做了83天。

为什么这么快就黄了？因为市场（全国）不答应啊！首先，最大的投资人兼"国际风投"——日本，发现袁老板想单干，立刻翻脸，联合其他列强施压。其次，公司内部的实力派股东们（云南的唐继尧、广西的陆荣廷等）直接掀桌子了，宣布"护国讨袁"，发起了护国战争。最关键的是，他赖以起家的"核心业务团队"北洋系内部也分裂了，他最能打的下属段祺瑞、冯国璋等都消极怠工，甚至暗中反对。

内外交困之下，袁世凯在1916年3月22日被迫取消帝制，试图回头继续当大总统。但人心散了，队伍不好带了。6月6日，他在全国一片骂声中，忧愤成疾（主要是尿毒症），一命呜呼。他临终前据说喃喃自语："他害了我。"这个"他"，有人说是怂恿他的杨度，有人说是伪造报纸的儿子，其实，害了他的，正是他那颗与时代潮流完全逆行的帝王之心。

二、 "护国"还是"互殴"？一场各怀鬼胎的战争

袁世凯一死，按理说"共和"该回来了吧？太天真了！护国战争这面大旗，从一开始就成分复杂，动机各异。

咱们可以把护国战争理解成一场"公司高管联合起来罢免想要搞家族世袭的CEO"的行动。但问题是，这些高管们，谁不想自己当CEO呢？

首先举旗的是云南的唐继尧和蔡锷。蔡锷是真心拥护共和的革命者，他带着病体，以"为国民争人格"的信念，率领护国军第一军入川苦战，堪称这场战争中唯一的"理想主义者之光"。但唐继尧呢？他更多是想趁机扩大滇系军阀的地盘和影响力，躲在后方给蔡锷使绊子、断补给的事可没少干。

广西的陆荣廷，这个绿林出身的"草头王"，加入护国阵营纯粹是因为袁世凯没给他足够的好处，而且他看准了袁大势已去，赶紧跳上"反袁"这辆顺风车，好名正言顺地占据两广。

其他各省的督军们，更是上演了精彩的"变脸"艺术。昨天还通电"恭祝皇帝陛下万寿无疆"，今天一看风向不对，立马通电"誓死捍卫共和，反对帝制自为"，速度之快，堪比川剧。

所以，护国战争在军事上其实规模不大，真正打的大硬仗不多。它更像是一场"政治表态赛"和"地盘确认会"。袁世凯死后，战争迅速平息，大家关心的不是如何重建共和秩序，而是："老董事长死了，公司总裁（大总统）位子空出来了，咱们怎么分（抢）？"

结果，副董事长黎元洪依法继任大总统，但实权掌握在国务院总理段祺瑞（北洋皖系首领）手里。中国进入了"府院之争"（总统府和国务院斗法）的新阶段，而地方上，那些在护国战争中亮过肌肉的"分公司老总"们，腰杆更硬了，开始真正意义上的"划地自肥"。一场更大的混乱，才刚刚拉开序幕。

三、 "分公司独立潮"：军阀割据的"战国地图"

如果要用一个词来形容1916年之后的民国，那就是"碎片化"。原来的"中华民国有限公司"，现在彻底变成了一个"军阀加盟连锁集团"，总部（北京政府）只是个名义上的"总店"，实际上对各家"加盟店"（各省军阀）毫无约束力。

咱们来画一张当时的"军阀割据地图"，您就明白了：

**东北片区：** 奉系军阀张作霖，人称"东北王"。他的地盘经营得最像独立王国，有兵工厂（沈阳兵工厂），有铁路，有港口，还有强大的空军（东北空军）和海军（东北海军）。老张是个精明的"地头蛇企业家"，在日、俄两大强邻的夹缝中左右逢源，把东北建成了自己的"家族企业"。

**华北片区：** 主要是北洋系分裂出来的直系（以冯国璋、曹锟、吴佩孚为首，控制直隶、江苏、湖北等地）和皖系（以段祺瑞为首，控制安徽、山东、福建等地）。这两家是"总部内斗"的主力，都自称是北洋正统，为了控制北京政府打得不可开交。

**东南片区：** 皖系、直系势力交错，还有像浙江的卢永祥、上海的何丰林等地方实力派。

**中南片区：** 桂系军阀陆荣廷（后为李宗仁、白崇禧）控制两广，滇系军阀唐继尧控制云南、贵州，湘系军阀谭延闿、赵恒惕控制湖南。这里是"地方特色品牌"聚集地。

**西北片区：** 陕系、甘系、宁系（马家军）等，情况更复杂，民族、宗教问题交织。

这些"军阀"是怎么经营自己地盘的？简单粗暴：第一，抓军队。有了枪杆子，就有了一切。他们疯狂扩军，士兵的军饷、装备全靠地盘内搜刮。第二，收税。各种苛捐杂税名目繁多，有的地方甚至预征了几十年后的田赋。老百姓苦不堪言，戏称"民国万税"。第三，种鸦片。这是最来钱的"特色产业"，很多军阀公开鼓励甚至强迫农民种植鸦片，以获取巨额军费，导致烟毒泛滥成灾。

他们之间是什么关系？今天你联合我打他，明天我联合他打你，毫无信义可言，全看利益。今天通电"结为兄弟"，明天就可能兵戎相见。北京那个"中央政府"，成了他们博弈的棋盘，大总统、总理像走马灯一样换，谁拳头硬，谁就能把自己的人扶上去。

四、 "东北王"的崛起：张作霖的"草莽枭雄"创业史

在众多军阀中，张作霖的故事最为传奇，堪称一部"从保险队长到陆海军大元帅"的草根逆袭史。

老张出身贫苦，当过兽医，干过土匪（他们自称"保险队"，就是收保护费保一方平安的民间武装）。但他脑子活，眼光毒。日俄战争时期，他在日俄两强之间巧妙周旋，两头拿好处，壮大了自己的队伍。清朝末年，他接受招安，成了正规军的军官。

民国以后，他凭借精明的政治手腕和强悍的军事力量，一步步统一东北，成了名副其实的"东北王"。他的管理风格非常"江湖"，讲义气，也重实利。他重用王永江等能吏发展经济、教育，让东北在乱世中相对稳定繁荣，工业化程度一度全国领先。但他也始终无法摆脱对日本关东军的依赖和妥协，在借款、修路等问题上留下了诸多隐患。

张作霖最大的特点就是"务实"和"野心"。他从不把"统一中国"挂在嘴上，而是埋头苦干，先把东北经营成铁桶一块。但他的目光从未离开过关内。他知道，要想坐稳江山，必须入主中原，拿到北京政府那个"正统"名分。因此，他与直系军阀的矛盾不可避免。

1920年的直皖战争，他联合直系打垮了皖系段祺瑞，分享了北京政权。但一山不容二虎，直系老大曹锟和吴佩孚，与这位关外来的"张大帅"，迟早要有一场决定华北归属的终极PK。

五、 "钞能力"选总统与"秀才将军"的末路：直系军阀的浮沉

说起直系，有两个核心人物不得不提：曹锟和吴佩孚。

曹锟，外号"曹三傻子"，其实一点也不傻，是个典型的机会主义者。他最大的"功绩"（或者说污点）是在1923年，用赤裸裸的贿赂手段，上演了一出"钞能力选总统"的丑剧。他给参加国会选举的议员们开出了高价：投他一票，当场奉上五千到一万大洋的支票（史称"贿选"）。果然，重赏之下必有"勇夫"，曹锟高票"当选"中华民国大总统。这件事彻底撕下了民国"民主共和"的遮羞布，遭到全国舆论的口诛笔伐，孙中山甚至在南方发起"讨曹"的北伐。

而直系的灵魂人物，其实是"玉帅"吴佩孚。吴佩孚是个另类，他出身秀才，饱读诗书，不抽不赌不嫖，号称"三不将军"，在腐败成风的军阀中堪称一股清流。他治军严谨，能征善战，在直皖战争和第一次直奉战争中大放异彩，一度控制了半个中国，被西方媒体称为"中国最强者"。

吴佩孚有传统士大夫的操守，他反对曹锟贿选，但又囿于"忠义"无法与之决裂。他有武力统一的雄心，但战略上树敌太多。更重要的是，他和他代表的直系，本质上仍是旧式军阀，脱离民众，内部腐化。他们的失败，不仅是军事上的，更是时代淘汰的必然。

六、 烽火连天京津路：两次直奉大战的"窝里斗巅峰"

直系和奉系，这两家当时中国最强的军事集团，为了争夺中央政权和华北地盘，在1922年和1924年，接连上演了两场规模空前的内战——直奉战争。

咱们可以把这理解为两家最大的"加盟店"，为了争夺"总店"（北京）的控制权和华北地区的"独家代理权"，展开的全方位商业（军事）火并。

**第一次直奉战争（1922年）：** 此时直系如日中天，吴佩孚指挥的直军装备精良，战术先进。而奉军虽然人多，但更像一支旧式武装，军纪涣散。结果，奉军在西线（长辛店、琉璃河一带）被吴佩孚打得大败，张作霖狼狈退回关外。战后，张作霖痛定思痛，花重金改革军制，建立现代化军校（东北讲武堂），从国外购买先进武器，大力建设海空军，实力迅猛提升。

**第二次直奉战争（1924年）：** 这是真正意义上的大决战。张作霖卷土重来，兵力更盛。而直系内部却出现了严重问题：第三军总司令冯玉祥，因为受吴佩孚排挤，早就心怀不满。在战争最关键的时刻，冯玉祥突然率部从热河前线秘密回师北京，发动政变（北京政变），软禁了贿选总统曹锟，彻底断了直系的后路。

前线直军闻讯士气崩溃，一败涂地。吴佩孚从天津乘船南逃，一代"秀才军阀"就此陨落。奉系大获全胜，张作霖的势力扩张到长江流域，达到了他人生的巅峰。

但这场胜利的果实，很快因为奉系内部的骄横和掠夺性政策（尤其在江南）而腐化。冯玉祥也没捞到好处，反而被张作霖和段祺瑞排挤。北京政权落入了张作霖和皖系残余势力组成的临时执政政府手中，段祺瑞再次出山担任"临时执政"，但这个政府比以往更加脆弱。

两次直奉大战，消耗了国家巨量的财富和生命，使华北民生凋敝。它没有带来统一，只是更换了坐在北京的那个最强的军阀。而南方，以广东为基地的孙中山及其后的广州国民政府，正在酝酿一场全新的、以革命党为内核的武力——国民革命军，一个新的风暴眼已经形成。

本章结语：

好了，咱们这壶茶喝到这里，是不是觉得嘴里有点发苦？这民国前十几年，简直是一锅越熬越糊的粥。从袁世凯的皇帝梦碎，到护国战争的名不副实，再到军阀割据的全面铺开，最后在直奉大战这种"顶级内耗"中达到混乱的高潮。

所谓的"中华民国"，在1912-1928年这十六年间，除了那块牌子，哪里还有一点现代民族国家的样子？它更像是一个被强行套上西装，内里却还是封建躯体，并且正在高速解体的畸形儿。中央权威荡然无存，法律成为一纸空文，政治沦为枪杆子的游戏，经济成为军阀的提款机，百姓生活在水深火热之中。

为什么会这么乱？根子在于辛亥革命的不彻底。它赶跑了一个皇帝，却没有摧毁滋生皇帝和军阀的社会土壤——分散的小农经济、强大的地方乡绅势力、列强的分而治之政策，以及最重要的是，没有建立起一支真正忠于国家而非个人的、有革命理想的新式军队。于是，旧王朝的军队（北洋新军）和各地的武装力量，顺理成章地转型为私人性质的军阀武装。

军阀混战的影响是极其深远的。它严重阻碍了中国现代化进程，让中国错过了第一次世界大战后发展的宝贵机遇期。它造成的分裂和贫困，为后来的外敌入侵（日本）埋下了伏笔。但另一方面，极度的混乱也松动了旧有的思想枷锁，新文化运动、五四运动正是在这样的背景下蓬勃兴起，为中国的重生寻找新的思想和道路。

所以，这一章我们看到的是一个"失败"的民国初年。但它并非毫无意义，它用血与火证明了一点：没有真正的国家统一和人民觉醒，任何形式的共和招牌，都不过是军阀们身上一件随时可以撕掉的外衣。

好了，茶凉了，咱们这一章就先讲到这儿。下一章，咱们的视线将转向南方，看看在一片混沌中，一股全新的、充满理想主义光芒的力量是如何艰难诞生，并最终点燃燎原之火的。那将是"从春秋到民国"最激昂澎湃的篇章之一。
