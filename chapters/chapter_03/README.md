第三章：史上最短的"霸道总裁"——大秦的公司兴亡史

![秦始皇兵马俑](https://img1.baidu.com/it/u=4069238038,2036650572&fm=253&fmt=auto&app=138&f=JPEG?w=800&h=1422)

各位看官，且为您重新沏上一壶热茶，今天咱们要聊的是一家在历史长河中昙花一现的"独角兽企业"。这家公司从创业到上市再到破产，前后不过十五年，堪称史上最短命的"世界五百强"。

一、 史上最强"收购狂魔"的创业之路

话说在公元前3世纪，商界新秀嬴政接过家族企业"秦国"的权杖时，这家公司还只是战国七雄中的一员。但这位年轻的董事长可不一般，他给自己定下了一个疯狂的目标：收购其他六家同行业公司！

您要问了，这位董事长凭啥这么自信？这就要说说秦国的核心竞争力了。早在商鞅担任"职业经理人"时，就给公司建立了一套堪称变态的管理制度。比如实行严格的KPI考核制度（军功爵制），业绩好的员工不仅能升职加薪，还能分房分地；业绩差的嘛，轻则降薪，重则"优化"。

更绝的是，嬴政董事长深谙"专业化分工"之道。他把公司业务分成两大板块：打仗和生产。打仗的专门打仗，种地的专心种地，各司其职，互不干扰。这就像现代企业的销售部和生产部，专业的人做专业的事，效率自然高。

![秦灭六国示意图](https://img1.baidu.com/it/u=1594572488,2771670140&fm=253&fmt=auto&app=138&f=JPEG?w=761&h=500)

从公元前230年收购韩国开始，这位"收购狂魔"开启了疯狂买买买模式。每收购一家公司，他就把对方的高管（贵族）调到咸阳总部"重点培养"，把对方的优质资产（土地、人口）纳入自己的管理体系。最搞笑的是收购齐国时，人家还在做着"和平共处"的美梦，结果秦军的收购团队直接就开到了家门口。

短短十年间，六家公司全部被吞并。公元前221年，嬴政董事长在咸阳总部召开了盛大的上市敲钟仪式，宣布"大秦集团有限公司"正式成立。这位三十九岁的董事长意气风发地宣布："从今天起，我不再是秦王，我要做皇帝！我要做始皇帝！"

二、 史上最严苛的"标准化管理"

公司上市了，难题也跟着来了。您想啊，收购了六家不同文化、不同制度的公司，怎么管理？嬴政董事长给出的解决方案是：标准化！

首先是文字标准化。以前七国各有各的书写方式，同一个字能有七八种写法。嬴政董事长一拍桌子："这还了得！公司内部文件都看不懂，还怎么高效办公？"于是他下令统一使用秦篆，相当于给全公司装上了统一的办公软件。

然后是交通标准化。您可能想不到，当时各国的车道宽度都不一样，这导致跨国贸易时得不停地换车。嬴政董事长大手一挥："全都给我统一成六尺！"这就好比现在的铁轨标准统一，大大提高了物流效率。

![秦驰道遗址](https://img0.baidu.com/it/u=1850824124,1928955502&fm=253&fmt=auto&app=138&f=JPEG?w=500&h=653)

最绝的是度量衡标准化。以前的秤、尺子、斗，每个国家都不一样。今天在齐国买一斤米，到楚国可能就只剩八两了。嬴政董事长下令制造标准度量衡器，分发到各地，相当于给全公司配发了统一的办公用品。

不过这位董事长的管理风格确实有点"霸道"。他推行的是典型的"狼性文化"，完不成KPI？重罚！违反公司规定？重罚！甚至连议论公司政策都要受罚。用现在的话说，这就是典型的"996是福报"的思维。

三、 那些年，秦始皇追过的"大项目"

要说这位始皇帝董事长，那可是个不折不扣的"项目狂人"。他上任后推动的几个大项目，放到现在都能上热搜。

首先是万里长城项目。这个项目的初衷是为了防范北方的竞争对手（匈奴）。嬴政董事长把原来各国修的边墙连接起来，形成了举世闻名的"大秦集团北部边防工程"。据说为了这个项目，动用了全国五分之一的劳动力，堪称古代版的"雄安新区"。

然后是阿房宫项目。这位董事长觉得咸阳总部太寒酸，配不上上市公司的身份，于是决定修建新的集团总部。这个项目有多夸张？司马迁在《史记》里写道："覆压三百余里，隔离天日。"这规模，比现在的苹果总部还要气派。

![秦始皇陵](https://img0.baidu.com/it/u=3125454817,889187994&fm=253&fmt=auto&app=138&f=JPEG?w=500&h=653)

最令人瞠目结舌的是秦始皇陵项目。这位董事长从13岁继承家业时就开始规划自己的"退休别墅"，动用了72万"建筑工人"，修了38年。里面的配套设施更是惊人：用水银做的江河湖海，用夜明珠做的日月星辰，还有那支闻名世界的"地下安保团队"——兵马俑。

这些大项目虽然彰显了公司实力，但也把员工们累得够呛。用现在的话说，这就是典型的"内卷"——老板一个比一个能折腾，员工一个比一个累。

四、 史上最著名的"企业文化整顿"

要说嬴政董事长最受争议的决策，非"焚书坑儒"莫属。这件事的起因，还得从一次公司内部座谈会说起。

公元前213年，嬴政董事长在咸阳宫举办年度座谈会。有个叫淳于越的"老员工"在会上发言，建议恢复分封制，说这是老祖宗传下来的好制度。这话可把董事长惹毛了："我们公司刚完成集团化改革，你就要开倒车？"

这时，公司的"人力资源总监"李斯站了出来。他说："董事长，现在有些员工总在背后议论公司政策，传播负能量。我建议把非技术类的书籍都收上来烧了，只保留技术手册和公司规章制度。"

![焚书坑儒](https://img2.baidu.com/it/u=423891447,2343056353&fm=253&fmt=auto&app=138&f=JPEG?w=500&h=500)

嬴政董事长一听，觉得很有道理，于是下令：除了医药、占卜、农业这些实用书籍，其他诸子百家的书统统烧掉！这就是著名的"焚书"事件。

更绝的是第二年，有两个"企业文化顾问"（方士）在背后说董事长坏话，还卷着项目经费跑路了。嬴政董事长大怒，下令在全公司范围内彻查"负能量传播者"，最后在咸阳处决了四百六十多个"问题员工"。这就是"坑儒"。

用现代企业管理的角度来看，这就是一次极端的"企业文化整顿"。只不过手段太过激烈，导致很多优秀人才寒了心。

五、 霸道总裁的突然"下线"

就在公司各项业务如火如荼进行时，一件意想不到的事情发生了。公元前210年，嬴政董事长在出差的路上突然"下线"了！

这件事说来颇具戏剧性。当时董事长正在巡视东南分公司，行至沙丘（今河北广宗）时突发急病。据小道消息称，这位工作狂董事长平时作息极其不规律，经常熬夜批阅奏章（据说每天要批阅一百二十斤竹简），还迷恋各种"保健品"（方士进献的仙丹），最终把身体搞垮了。

在生命的最后时刻，这位霸道总裁写下一封"任命书"，让长子扶苏接任董事长。但这时，一个关键人物出场了——董事长秘书赵高。

![沙丘之变](https://img2.baidu.com/it/u=2471694496,3800289704&fm=253&fmt=auto&app=120&f=JPEG?w=889&h=500)

赵高这个人心思缜密，他找到公司二公子胡亥和"运营总监"李斯，进行了一次秘密会谈。他对李斯说："老李啊，扶苏要是上位，肯定重用蒙恬，到时候还有你什么位置？"这话正好戳中了李斯的软肋。

三人一合计，决定隐瞒董事长去世的消息，把遗体放在车上继续赶路。为了掩盖气味，还在车上装了一车咸鱼。这大概是史上最味儿的权力交接了。

他们伪造了一份新的"任命书"，让胡亥继任董事长，还假传圣旨逼扶苏自尽。等回到咸阳总部，才公布董事长死讯。就这样，秦二世胡亥稀里糊涂地坐上了董事长的位置。

六、 史上最荒唐的"内部会议"

胡亥这个新任董事长，用现在的话说就是个"佛系青年"。他上任后的第一件事就是把公司大小事务都交给赵高处理，自己专心享乐。

赵高这下可得意了，从董事长秘书一跃成为实际上的"CEO"。但他心里还是不踏实，因为朝中还有很多老员工不服他。于是，他想出了一个馊主意来测试人心。

一天早会上，赵高牵来一头鹿，对胡亥说："董事长，我最近得到一匹好马，特意献给您。"胡亥一看就乐了："爱卿，你眼睛是不是出问题了？这分明是头鹿啊！"

![指鹿为马](https://img2.baidu.com/it/u=3087356170,3214557193&fm=253&fmt=auto&app=138&f=JPEG?w=500&h=667)

赵高却一本正经地说："这确实是马，不信您问问其他同事。"于是他转向在场的各位高管："各位说说，这到底是鹿还是马？"

这下可热闹了。有些机灵的高管立即附和："确实是马，好马！"有些耿直的老臣坚持说："明明是鹿！"还有些墙头草支支吾吾不敢表态。

会后，赵高就把那些说"鹿"的人都记在了小本本上，没过多久就找各种理由把他们"优化"掉了。这就是史上著名的"指鹿为马"事件，堪称最荒唐的公司内部会议。

经此一事，公司内部再也没有人敢说真话。大家都在忙着站队、表忠心，真正干实事的人越来越少。

七、 前员工的"维权行动"与公司破产

就在总部上演"指鹿为马"的闹剧时，基层员工们已经忍无可忍了。最直接的导火索，是公司推行的"严苛考勤制度"。

公元前209年，一队员工（戍卒）在项目经理陈胜、吴广的带领下，前往北方出差。结果遇上大雨，道路不通，眼看就要迟到。按照公司规定，迟到是要掉脑袋的。

陈胜和吴广一合计："横竖都是死，不如反了他娘的！"于是他们杀掉了随行的HR（将尉），召集所有员工开动员大会。陈胜站在高处，说出了那句流传千古的名言："王侯将相，宁有种乎！"意思是：董事长怎么了，谁规定我们就不能当董事长？

![大泽乡起义](https://img1.baidu.com/it/u=276598559,3518056939&fm=253&fmt=auto?w=1280&h=800)

这场"维权行动"很快演变成了全国性的"罢工潮"。原来六国分公司的老员工们也趁机起来"维权"，要求恢复原来的公司架构。

更糟糕的是，这时出现了两位实力强劲的"并购专家"——项羽和刘邦。项羽是原楚国分公司的"太子爷"，自带贵族光环；刘邦则是个草根出身的"销售经理"，特别会来事。

胡亥董事长听说各地都在"维权"，居然还不相信。他对赵高说："不可能，绝对不可能！我们公司的员工满意度一直很高。"赵高也顺着他说："都是些小打小闹，不成气候。"

等到项羽在巨鹿大破秦军主力，刘邦带队直扑咸阳总部时，一切都晚了。赵高眼看大势已去，干脆派人把胡亥董事长也给"优化"了。

八、 最后的清算与历史教训

公元前207年，刘邦的"并购团队"开进咸阳。此时继任的秦王子婴已经无力回天，只好捧着公司的公章（玉玺）和营业执照（符节）在路边投降。

仅仅存在了十五年的大秦集团有限公司，就这样宣告破产清算。这家曾经不可一世的巨无霸企业，最终落得如此下场，实在令人唏嘘。

复盘这家公司的兴衰，我们可以发现几个致命问题：首先是企业文化太过严苛，把员工逼到了对立面；其次是接班人选择失误，让一个"佛系青年"执掌大权；最重要的是，公司在高速扩张后没有及时调整管理策略，仍然沿用创业期的那套高压手段。

![项羽刘邦进军咸阳](https://img1.baidu.com/it/u=2776548274,1517836080&fm=253&fmt=auto&app=138&f=JPEG?w=500&h=501)

有意思的是，完成最终"并购"的刘邦深刻吸取了秦朝的教训。他上任后推行的是"宽松管理"，减轻员工负担，这才有了后来持续四百多年的汉朝集团。

本章结语：
您看，大秦这家公司就像现在的某些互联网巨头，靠着狼性文化和严格管理迅速崛起，却又因为同样的原因快速陨落。秦始皇这个"霸道总裁"确实能力出众，但他忘了最重要的一点：公司不是他一个人的，而是所有员工的。

他那些伟大的梦想——统一文字、统一度量衡、修建长城——确实功在千秋，但实现梦想的方式却太过激进。用现在的话说，就是太"卷"了，把所有人都卷得筋疲力尽。

最讽刺的是，他为了防止员工"维权"，把民间的兵器都收上来熔铸成了十二个金人，结果陈胜吴广拿着木棍就把事情闹大了。这说明什么？说明真正可怕的不是员工手里有什么武器，而是他们心里有什么想法。
