第三十一章："表面繁荣"的"盛世"——康雍乾的"最后辉煌"

__康雍乾三帝画像__

各位看官，且泡上一壶上好的普洱，咱们这就进入一个金光闪闪、却又迷雾重重的时代。今天要聊的，是号称“中国古代最后一个盛世”的康雍乾时期。您可能要问，盛世就盛世，为啥还要打引号？唉，因为这事儿啊，就跟朋友圈里那些加了十八层滤镜的精修图一样，乍一看繁花似锦，仔细一瞧，全是“照骗”（照片骗局）。表面上GDP世界第一，人口突破三亿，疆域空前辽阔，但骨子里，这套“超大型传统家族企业”的管理模式，已经开始跟不上世界“工业革命”的版本更新了。咱们这就剥开盛世的糖衣，看看里面藏着什么。

一、 康熙：上任就遇“地狱模式”，平三藩如同“清理门户”

话说公元1661年，北京城。一个刚满8岁的小男孩，爱新觉罗·玄烨，坐在了紫禁城那把对他来说还有点大的龙椅上。这就是后来的康熙帝。他接手的，可不是什么“太平公司”，而是一个成立不到20年、内部矛盾重重的“新创企业”。外面有南明残部、台湾郑氏（郑成功家族）这些“竞争对手”虎视眈眈；里面呢，还有三个手握重兵的“开国元勋”兼“大区总裁”——平西王吴三桂（管云南）、平南王尚可喜（管广东）、靖南王耿精忠（管福建）。这三个人，史称“三藩”，他们可不是普通的地方官，那是拥有独立军队、财政、人事任免权的“国中之国”，妥妥的“分公司独立王国”。

康熙这小朋友，年纪虽小，心眼可不小。他隐忍了八年，一边读书学习（史载他每天学习时间超过10小时，堪称“卷王”），一边暗中观察。到了20岁亲政时，他觉得时机差不多了。正好，平南王尚可喜年纪大了，想退休回辽东老家，上书请求“撤藩”（撤销藩王编制）。康熙一看，机会来了！他大手一挥：准了！不仅准了尚可喜，还顺带“好意”地问吴三桂和耿精忠：“二位老同志要不要一起撤了，回来享享清福啊？”这招“以退为进”，表面是关心，实则是逼宫。

![吴三桂起兵](https://img0.baidu.com/it/u=3811117802,2850644552&fm=253&fmt=auto&app=138&f=JPEG?w=717&h=500)

果然，老狐狸吴三桂不干了。他在云南经营多年，兵强马壮，早就把自己当“云贵王”了。1673年，吴三桂打出“反清复明”的旗号（虽然他自己就是明朝的“叛徒”，这操作堪称“又当又立”的典范），起兵造反。耿精忠、尚可喜的儿子尚之信（这哥们后来也反了）纷纷响应，一时间，大半个南方都乱了套。这就是著名的“三藩之乱”。

朝廷里一片恐慌，很多大臣主张妥协，甚至有人建议杀了主张撤藩的大臣，向吴三桂道歉。年仅20岁的康熙却展现出了惊人的魄力，他力排众议，坚决主战：“撤亦反，不撤亦反，等他们准备好了再反，不如现在就打！”他亲自调兵遣将，启用了一大批汉人将领，比如图海、赵良栋、王进宝等。这场仗一打就是八年，过程极其惨烈，双方拉锯，清军一度非常被动。但康熙的战略定力极强，他采取分化瓦解、重点打击的策略，集中力量先打垮了耿精忠和尚之信，最后全力围攻吴三桂的老巢。

1681年，清军攻入昆明，吴三桂的孙子吴世璠自杀，历时八年的三藩之乱终于平定。这场胜利，相当于中央集团成功“清理门户”，收回了地方“分公司”的财权、军权和人事权，实现了真正意义上的“大一统”。康熙的威信也由此达到顶峰，为他后来解决台湾问题（1683年施琅平台）、抗击沙俄（雅克萨之战，签订《尼布楚条约》）、平定噶尔丹（三次亲征）等一系列“KPI”打下了坚实基础。不过，您可能不知道，这位雄才大略的皇帝，还是个“科学发烧友”，他本人认真学习过数学、天文、地理甚至西医，还曾亲自用金鸡纳霜（奎宁）治好了自己的疟疾。可惜，他的这些兴趣，仅仅停留在个人爱好层面，并未转化为推动国家科技变革的动力。

二、 雍正：史上最勤勉“CEO”，“摊丁入亩”与“火耗归公”的财税大改革

1722年，康熙老爷子在畅春园驾崩。一场惊心动魄的“夺嫡大戏”落下帷幕，皇四子胤禛胜出，这就是雍正皇帝。关于他继位的合法性，民间野史（比如“传位于四子”改“传位十四子”的段子）传得神乎其神，咱们暂且不表。但有一点可以肯定，雍正接手的，是一个外表光鲜、内里却有些“虚胖”的帝国。

他爹康熙在位61年，虽然功绩卓著，但晚年不免有些“仁慈”过了头。官员贪污成风，国库银子看着多，但一查账，好多都是“应收账款”，实际库存空虚。土地兼并严重，有钱有势的士绅利用“人头税”（丁税）的漏洞，把税收负担转嫁给贫苦农民，导致大量农民逃亡，社会矛盾加剧。用现在的话说，就是“贫富差距拉大，基层征税困难，中央财政吃紧”。

__雍正批阅奏折__

雍正一上台，就展现出了和他爹完全不同的风格。康熙是“宽仁”，雍正则是“严苛”。他是中国历史上出了名的“工作狂”，平均每天只睡4个小时，一年只在生日那天给自己放一天假。现存的朱批奏折有数万件，他经常在奏折上写下比原文还长的批语，事无巨细，亲自过问。他给自己定的KPI就俩字：“整顿”。

他搞了两项核心改革，堪称清朝中期的“财税系统大升级”。
第一项叫“摊丁入亩”。以前收税分两部分：土地税（田赋）和人头税（丁银）。家里地再多，交的田赋是固定的，但人头税是按男丁数量收。这就导致地主家丁多，但通过隐瞒、投靠（让佃农挂在自己名下逃避丁税）等方式，实际交的税可能比贫农还少。雍正把这个“丁银”摊到“田亩”里去，地多的人多交，地少的人少交，没地的人不交。简单说，就是从“按人头收税”变成“按财产（土地）收税”。这大大减轻了无地、少地农民的负担，抑制了土地兼并，也增加了国家税收的稳定性。相当于把税基从流动的“人”，锚定在了固定的“地”上。

第二项叫“火耗归公”和“养廉银”。这得解释一下。古代收税，老百姓交的是碎银子，官府要熔铸成统一规格的银锭上交国库。熔铸过程中会有损耗，这叫“火耗”。地方官就以此为借口，在法定税收之外，多收一笔“火耗银”来弥补“损耗”。实际上，“火耗”往往远超实际损耗，多出来的部分就进了官员自己的腰包，成了公开的“灰色收入”。雍正下令，火耗银子由地方政府统一征收，但必须全部上交省库，再由省里根据各级官员的职务、地方公务开销等情况，发一笔“养廉银”作为合法津贴和办公经费。这招“高薪养廉”+“收支两条线”，一定程度上遏制了基层乱收费，也保障了官员的基本待遇（当然，贪腐不可能根除）。

雍正的改革，触动了既得利益集团（主要是官僚和大地主）的蛋糕，所以他被士大夫阶层黑得很惨，民间也流传着许多关于他“刻薄寡恩”、“弑父逼母”的谣言。但他这套“精细化管理系统”，确实给清朝这台老旧的机器打了一剂强心针，国库从康熙晚年的不到800万两，增加到雍正末年的6000多万两，为接下来的“乾隆盛世”攒下了厚厚的家底。他就像一个严厉的财务总监，通过强硬手段，让公司“回血”了。

三、 乾隆：十全老人的“KPI”狂欢与审美“暴发户”

1735年，雍正去世，皇四子弘历即位，这就是乾隆皇帝。他可能是中国历史上最“幸运”的皇帝之一：爷爷康熙打好了江山基础，爸爸雍正攒够了钱袋子，他25岁登基，接手的是一个国库充盈、政局稳定、疆域广袤的超级帝国。而且他身体倍儿棒，一口气活了89岁，实际执政63年（外加3年太上皇），是中国历史上实际掌权时间最长的皇帝。

乾隆的自我感觉非常良好，他的人生追求就是“十全武功”和“文化盛世”。所谓“十全武功”，是指他自诩在位期间打赢的十场大仗：两次平定准噶尔、一次平定回部、两次平定大小金川、一次镇压台湾林爽文起义、一次征缅甸、一次征安南（越南）、两次抗击廓尔喀（尼泊尔）入侵。听着挺唬人吧？但咱们仔细掰扯掰扯：打准噶尔和回部（新疆），确实是开疆拓土，巩固西北边防，功绩显著。但打大小金川（四川少数民族土司），前后耗银7000多万两，死伤数万，就打两个巴掌大的地方，性价比极低，纯属“为了打而打”，刷战绩。征缅甸、打安南，基本都是劳师远征，没占到什么便宜，最后对方“请和”了事，乾隆就自己宣布胜利，列入“武功”。这“十全武功”里，水分不小，很多是为了凑数，满足他“十全老人”（他晚年自封的号）的虚荣心。

__乾隆南巡图__

除了打仗，乾隆还特别喜欢搞文化工程。他主持编纂了中国历史上最大的丛书《四库全书》，收录了3500多种书籍，7万多卷，堪称文化盛举。但是！编书的过程中，也伴随着严厉的“文字狱”和图书审查。大量不符合清朝统治思想的书籍被禁毁、篡改，造成的文化损失难以估量。这就像一边建最大的图书馆，一边又烧掉不少“不合规”的书，功过实在难以简单评说。

乾隆的审美，也一直被后人吐槽。他特别喜欢在各种古董字画上题字、盖章，看到王羲之的《快雪时晴帖》要题个“神”，看到黄公望的《富春山居图》要盖上几十个章，被网友戏称为“爱新觉罗·牛皮癣”、“弹幕鼻祖”。他喜欢的瓷器风格，也是色彩浓艳、纹饰繁复，被现代许多人认为充满了“暴发户”式的审美趣味。他六次南巡，排场极大，耗费巨资，虽然有一定的政治巡视目的，但更多的是为了满足个人游乐和炫耀“盛世”气象。

在乾隆统治的前中期，清朝的国力确实达到了鼎盛。人口突破三亿，占世界三分之一；GDP总量世界第一；疆域达到1300多万平方公里，是今天的1.35倍。但这一切繁华，都建立在传统农业经济和庞大人口红利的基础上。就在乾隆陶醉于“天朝上国”的迷梦时，世界正在发生翻天覆地的变化。1765年，英国哈格里夫斯发明了珍妮纺纱机，工业革命拉开序幕；1776年，美国独立；1789年，法国大革命爆发……而紫禁城里的乾隆，还在忙着写他的四万多首诗（数量顶得上《全唐诗》，质量嘛……你懂的），欣赏着江南进贡的“苏绣”和“淮扬菜”。

四、 文字狱：盛世下的“思想冷冻柜”

聊完了三位皇帝的“政绩”，咱们得说说这个盛世里一个让人脊背发凉的角落——文字狱。文字狱这东西，历朝历代都有，但清朝，特别是康雍乾三朝，把它发展到了“登峰造极”的程度。如果说康熙朝的文字狱还有点“针对性”（主要针对那些怀念明朝、私下里搞“反清复明”宣传的人），那么到了雍正、乾隆朝，文字狱就彻底“内卷”和“扩大化”了，变成了一种制造恐怖气氛、禁锢思想的统治工具。

咱们讲几个“离谱他妈给离谱开门——离谱到家了”的例子。
康熙朝有个叫“明史案”的大案。浙江富户庄廷鑨买了一部明朝遗民写的《明史》稿本，自己出钱修补刊印。书里用了南明年号，称清军为“夷寇”。这事被人告发，康熙大怒。此时庄廷鑨已死，结果被刨坟戮尸！他的父亲、兄弟、子侄，以及为此书写序、校对、刻印、卖书甚至买书的人，全部被处死，牵连上千人。这就像今天有人出版了一本“不合规”的书，不仅作者全家遭殃，连印刷厂工人、书店老板、买书的读者都要被抓，恐怖程度可想而知。

雍正朝更绝。有个考官出科举试题，用了《诗经》里的“维民所止”四个字。结果有人告发说，“维”和“止”两个字，是“雍正”二字去了头，暗示要砍雍正的头！雍正一看，这还了得？尽管出题人百般解释这只是经典里的普通句子，但最终还是被关进大牢，病死狱中，家属流放。这就属于典型的“欲加之罪，何患无辞”，靠汉字拆字来罗织罪名，想象力“丰富”得让人无语。

![清代文字狱档案](https://img1.baidu.com/it/u=426241112,759888446&fm=253&fmt=auto&app=138&f=JPEG?w=500&h=500)

到了乾隆朝，文字狱达到了巅峰，数量超过130起。有个叫徐述夔的举人，写过两句诗：“明朝期振翮，一举去清都。”意思是明天早晨展翅高飞，一下飞到清都（天帝居所）。这本来是个很普通的励志诗句。但被人举报说，“明朝”暗指“明朝”，“去清都”是“去掉清朝都城”。乾隆批示：“罪大恶极！”徐述夔已死，照样开棺戮尸，子孙处斩。连已去世多年的沈德潜（乾隆的御用诗人，生前备受荣宠）因为给徐述夔的诗集写过序，死后也被追责，剥夺谥号，砸毁墓碑。

在这种高压下，知识分子们还敢研究思想、探讨时政吗？大家只能埋头故纸堆，搞搞考据、训诂（研究古文字、古文献），这就是所谓的“乾嘉学派”。学问做得再细，也不敢越雷池一步。整个思想界被装进了一个巨大的“冷冻柜”，万马齐喑。当西方的思想家在讨论“自由”、“平等”、“人权”、“三权分立”的时候，中国的读书人还在为“茴香豆的‘茴’字有几种写法”而绞尽脑汁。思想的停滞，是一个文明衰落最危险的信号。

五、 闭关锁国：自己关上与世界接轨的“大门”

康雍乾时代，还有一个对后世影响极其深远的国策，那就是“闭关锁国”。这个词儿大家耳熟能详，但具体怎么个“闭”法，咱们得说道说道。

其实清朝初年，为了对付东南沿海的郑成功势力，实行了严厉的“海禁”，片板不许下海。康熙在平定台湾后，于1684年一度开放了“四口通商”（广州、漳州、宁波、云台山），对外贸易有所恢复。但到了乾隆皇帝这里，他觉得“天朝物产丰盈，无所不有，原不藉外夷货物以通有无”。意思是，咱家里啥都有，根本不需要跟外国佬做生意。

1757年，乾隆一道圣旨，关闭了漳州、宁波、云台山三个海关，只留广州“一口通商”。而且，外国商人来广州做生意，不能直接跟中国老百姓接触，必须通过官方指定的“十三行”商人进行。外国商人只能住在广州城外的“夷馆”里，活动受限，也不能带家属，不能学中文。管理外国商人的规章多如牛毛，比如《防范外夷规条》等等。

__广州十三行__

这不仅仅是限制贸易。更重要的是，它关闭了中外文化交流的渠道。西方的科学技术、思想观念、国际形势，很难通过这唯一的、被严格管控的窗口传入中国。朝廷和绝大多数士大夫，对世界正在发生的巨变一无所知，或者嗤之以鼻，认为那是“奇技淫巧”。

1793年，英国派马戛尔尼使团访华，带来了天文望远镜、地球仪、蒸汽机模型、榴弹炮、战舰模型等最能代表当时英国科技水平的礼物，希望与清朝建立正式外交关系，扩大贸易。但清朝官员把这些礼物视为“贡品”，关心的是马戛尔尼是否愿意行“三跪九叩”大礼。乾隆皇帝在给英国国王的“敕谕”中写道：“天朝德威远被，万国来王，种种贵重之物，梯航毕集，无所不有……然从不贵奇巧，并无更需尔国制办物件。”意思是，你们这些东西，我们都不稀罕，我们啥都有，你们就乖乖听话就行。一次重要的中西对话机会，就这样在“礼仪之争”和“天朝上国”的傲慢中化为泡影。

关上了门，也就关上了眼睛和耳朵。当清朝上下还沉浸在“盛世”的自我陶醉中时，一场即将改变中国命运的巨变，已经在海上酝酿。

六、 盛世背后的“蛀虫”：腐败、人口与生态危机

金光闪闪的盛世外表下，蛀虫早已滋生。首先是腐败，这是帝国肌体上最顽固的毒瘤。康熙晚年，官员贪污已蔚然成风，最大的贪官明珠、索额图权倾朝野。雍正虽然严厉反腐，但人亡政息。到了乾隆朝，腐败达到了惊人的程度。乾隆本人好大喜功，喜欢排场，下面的人就投其所好，铺张浪费。最大的贪官和珅，通过乾隆的宠信，聚敛了巨额财富，据说家产相当于清政府十几年财政收入，号称“和珅跌倒，嘉庆吃饱”。各级官吏层层盘剥，“三年清知府，十万雪花银”是常态。腐败严重侵蚀了国家财政和行政效率。

其次是人口爆炸带来的压力。清朝前期社会相对稳定，引进高产作物（玉米、番薯、土豆），使得人口从明末的不到一亿，猛增到乾隆末年的三亿左右。人口是红利，也是负担。耕地增长远远赶不上人口增长，人均耕地面积急剧下降，大量农民失去土地，成为流民或佃农，生活极端贫困。社会就像一个被不断吹大的气球，内部压力越来越大。

__流民图__

再者是生态环境的破坏。为了养活庞大人口，人们不断开垦荒地，围湖造田，向山区、林地要粮。这导致森林植被破坏，水土流失加剧，河流湖泊淤塞，自然灾害（水灾、旱灾）愈发频繁。黄河、长江的泛滥在乾隆中后期明显增多，治理成本高昂，效果却很差。

这些危机，都被“盛世”的繁华景象所掩盖。朝廷的财政收入主要来自农业税，工商业税占比极低，无法开辟新税源。当传统农业社会的承载力达到极限时，整个系统就变得非常脆弱，经不起任何大的冲击。

七、 错失的机遇：当中国沉睡时，世界在狂奔

让我们把镜头拉远，看看同一时期的地球另一端在发生什么。
1687年（康熙二十六年），牛顿发表《自然哲学的数学原理》，经典力学大厦奠基。
1765年（乾隆三十年），英国开始工业革命。
1776年（乾隆四十一年），美国发表《独立宣言》，亚当·斯密出版《国富论》（主张自由贸易）。
1789年（乾隆五十四年），法国大革命爆发，发表《人权宣言》。
1796年（乾隆退位次年），英国医生琴纳发明牛痘接种法预防天花。

与此同时，中国的知识分子在干嘛？在考科举，读四书五经，写八股文。中国的工匠在干嘛？在做精美的瓷器、丝绸，技术是祖传的手工秘诀。中国的皇帝在干嘛？在避暑山庄接见蒙古王公，在紫禁城欣赏西洋钟表（只当是玩具），在批阅那些歌颂“太平盛世”的奏章。

科技革命的浪潮、思想启蒙的光芒、政治制度的变革……所有这些近代化的关键要素，都被那堵“闭关锁国”的墙挡在了外面。清朝拥有当时世界上规模最庞大的市场、最众多的人口、最稳定的中央政权，本最有条件应对变局，甚至参与并引领变革。但它选择了转过身去，沉溺于旧日的辉煌。

所谓的“康雍乾盛世”，就像一台运行了太久、从未升级过操作系统的老电脑。它的硬件（疆域、人口）在当时看来确实顶级，它的软件（儒家治国理念、中央集权制度）也曾非常高效。但是，当世界已经进入“工业革命3.0”版本时，它还在用“农业文明1.0”的思维和代码。表面的繁荣，是旧系统超负荷运转的最后辉煌，也是系统崩溃前“回光返照”式的亮光。

本章结语：
所以，各位看官，您现在明白为什么这“盛世”要打上引号了吧？康雍乾三代，确实打造了一个在传统框架内登峰造极的帝国：疆域之广，前代少有；人口之众，世界第一；中央集权，空前强化。康熙的雄才、雍正的勤政、乾隆的自信（或者说自负），共同塑造了这个“最后的古典盛世”。

然而，这个盛世的底色，是思想的禁锢（文字狱）、是眼界的封闭（闭关锁国）、是制度的僵化（强化皇权与官僚体系）、是创新的停滞。它把所有精力都用于维护内部的稳定和传统的延续，对外部世界翻天覆地的变化选择了无视和排斥。当帝国上下都在为“十全武功”和“万国来朝”的场面而陶醉时，历史的指针已经悄然拨向了另一个方向。这个盛世，没有孕育出走向现代社会的动力，反而像一层厚厚的包浆，将中国包裹在旧时代的壳里。它绚烂的夕阳，映照出的不是新生的黎明，而是漫长黑夜来临前的最后余晖。

好了，茶凉了，咱们这一章关于“表面繁荣”的盛世就先讲到这儿。下一章，我们将迎来“天朝上国”的第一次剧烈震动——鸦片战争。那一声炮响，将彻底惊醒这个沉睡了太久的帝国迷梦，让我们看看，这艘古老的巨轮，将如何驶向惊涛骇浪的近代。
